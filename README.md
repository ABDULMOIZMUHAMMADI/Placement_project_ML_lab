Introduction
This project is about predicting whether a student will get placement or not by using Machine Learning. The system checks different student data like CGPA, skills, internships, communication skills and projects.
We are using Logistic Regression algorithm for prediction.

Problem Statement
Many students dont know which things help them in getting placement. Universities also need a system that can predict placement chances of students.
This project will predict:
•	Placed
•	Not Placed

Objectives
•	To make a placement prediction system
•	To check which factors affect placement
•	To train and test ML model
•	To find accuracy of the model

Dataset
The dataset contains student information like:
•	CGPA
•	Skills
•	Internships
•	Communication
•	Projects
•	Placement status
Dataset has almost 1000 records.

Methodology
1.	Generate and collect dataset
2.	Preprocess the data
3.	Train the model using Logistic Regression
4.	Test the model
5.	Check accuracy score
6.	Save the model using Pickle

Tools and Technologies
•	Python
•	Pandas
•	NumPy
•	Scikit-learn
•	Jupyter Notebook(I used collab)
Output
The system will take student details and predict if the student can get placement or not.  
Conclusion
This project shows how Machine Learning can help in predicting student placements. It also helps in understanding ML concepts like training, testing and prediction in a simple way.
